# 03 — Space Shooter

Shooter vertical arcade en **Godot 4.2+**. Mover, disparar, esquivar enemigos y sumar puntos.

## Controles

- **WASD / flechas** — mover nave
- **Espacio** — disparar
- **R** — reiniciar

## Cómo abrir

Godot 4.2+ → Import → `project.godot` → F5

## Estructura

```
03_SPACE_SHOOTER/
  scenes/main.tscn, bullet.tscn, enemy.tscn
  codigo/game.gd, player.gd, bullet.gd, enemy.gd
```

## Aprendizajes

- Spawn procedural de enemigos
- `Area2D` para proyectiles
- Colisiones por capas
- Game over y puntuación
