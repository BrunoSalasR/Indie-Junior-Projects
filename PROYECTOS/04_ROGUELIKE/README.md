# 04 — Roguelike

Roguelike por turnos en **Godot 4.2+**. Explora dungeons generados, combate enemigos al chocar con ellos y avanza de piso en piso.

## Controles

- **WASD / flechas** — mover (un tile por turno)
- **R** — reiniciar

## Objetivo

Llega al tile dorado (salida) sin quedarte sin HP. Cada piso tiene más enemigos y laberintos más densos.

## Cómo abrir

Godot 4.2+ → Import → `project.godot` → F5

## Estructura

```
04_ROGUELIKE/
  scenes/main.tscn
  codigo/game.gd, dungeon.gd
```

## Aprendizajes

- Generación procedural de mapas
- Movimiento en grid y turnos
- IA simple de enemigos
- Progresión por pisos
