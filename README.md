# Indie Junior Projects

Godot 4 learning portfolio — small games from Pong to a turn-based dungeon crawler. Each folder under `PROYECTOS/` is a standalone project with its own `project.godot`.

| # | Project | Level | What it teaches |
|---|---------|-------|-----------------|
| 01 | [Classic Pong](PROYECTOS/01_PONG/) | Easy | Two-player input and collision |
| 02 | [2D Platformer](PROYECTOS/02_PLATAFORMERO_2D/) | Easy | Jump, double jump, basic enemies |
| 03 | [Vertical Space Shooter](PROYECTOS/03_SPACE_SHOOTER/) | Easy | Spawning, shooting, arcade loop |
| 04 | [Turn-Based Roguelike](PROYECTOS/04_ROGUELIKE/) | Easy | Grid movement and floor progression |
| 05 | [Nexos](https://github.com/Brunich/Nexos) | Advanced | Full fan RPG — separate repo |

> **Nexos** moved to its own repository. The copy under `PROYECTOS/05_NEXOS/` here is a snapshot; use [Brunich/Nexos](https://github.com/Brunich/Nexos) for the full game.

## Run any project

1. Install [Godot 4.2+](https://godotengine.org/)
2. Open the `project.godot` inside the project folder
3. Press **F5**
